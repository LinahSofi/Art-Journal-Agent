# Art Journal Agent

A personal AI-powered art journal where you can describe your artwork ideas and receive thoughtful critique, creative suggestions, and inspiration — powered by Claude AI.

## Features

- Describe any artwork idea, concept, or piece you're working on
- Get structured feedback: critique, suggestions, and an inspiring thought
- Streaming responses for a natural, conversational feel
- Beautiful dark UI designed for creatives

## Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Node.js, Express
- **AI:** Anthropic Claude API (claude-sonnet)

## Getting Started

1. Clone the repo
git clone https://github.com/LinahSofi/Art-Journal-Agent.git
2. Install dependencies
npm install
3. Create a `.env` file and add your Anthropic API key
ANTHROPIC_API_KEY=your_key_here
4. Start the server
node server.js
5. Open your browser at `http://localhost:3000`
